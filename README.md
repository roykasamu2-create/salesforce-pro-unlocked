## 🛠️ Example Profile Configuration  
Create a `profile.yml` in your working directory:  
```yaml
profile:
  name: "Sales Rep - EMEA"
  languages: ["en", "fr", "de"]
  automation_rules:
    - event: "lead_assigned"
      action: "send_followup_email"
      delay: "1h"
  integration:
    - api: "HubSpot"
      sync: "contacts"  
  ui_preferences:
    theme: "dark"
    widgets: ["pipeline", "forecast"]
```
